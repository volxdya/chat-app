<h1>Запуск и установка зависимостей</h1>

<h2>Backend</h2>

Нужно иметь установленный PostgreSQL и Redis, если вы запускаете проект без докера.

```
npm i
npm run start:dev
```

<h2>Docker</h2>

```
docker-compose build
docker-compose up
```
